# 枢纽：天下之道

> 基于施展[《枢纽：3000年的中国》](https://book.douban.com/subject/27602889/)的历史策略游戏

**在线试玩** : https://orangeviolin.github.io/pivot-game/

---

## 游戏简介

跨越三千年，在七大空间中做出历史抉择。你的每个选择都会引发连锁反应——中原的繁荣可能让草原联盟，海洋的开放可能带动西南发展。

**核心问题：何谓中国？**

这不是"选一个国家打天下"的游戏。你要做的是理解一个超大规模文明体系如何运转——多元共生，而非单一统治。

## 游戏内容

| 维度 | 数量 |
|------|------|
| 历史时代 | 4 纪（封建 → 豪族 → 平民 → 现代） |
| 历史事件 | 36 个（每纪 8 常规 + 1 转型/终局） |
| 决策选项 | 108 个（每事件 3 选） |
| 空间区域 | 7 个（中原/草原/西域/过渡地带/高原/西南/海洋） |
| 结局 | 5 种 |

## 五种结局

| 结局 | 达成条件 |
|------|---------|
| **天下枢纽** | 七大空间均衡繁荣 |
| **中原帝国** | 中原极强，其他衰落 |
| **海洋时代** | 海洋贸易主导 |
| **守成之局** | 不好不坏，平庸之选 |
| **天下大乱** | 各空间分崩离析 |

## 设计理念

书中核心论点直接融入游戏机制：

- **超大规模性**：7 个区域 × 4 维属性（兴/武/文/商），系统复杂但每次只需做一个选择
- **多元共生**：忽视任何空间都会被惩罚，均衡发展获得加成
- **连锁反应**：决策影响在区域间传播，体现"牵一发动全身"
- **历史大循环**：4 个时代有不同机制规则，转型事件是关键节点
